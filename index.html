Adventures in Ruby

Before applying to Founders and Coders, I was learning Ruby. Here are some of the things I worked on:

Alarm clock

I made this after asking a friend what programs he needed on his computer. It didn’t take me much more than an hour, but I enjoyed making it. I like how the project, in combination with my modest abilities, forced me to use and abuse Ruby’s kernel features, like while-loops and the bell terminal output. The alarm clock is a bit ‘hacky’, and for some reason, that makes me proud.

Web Guesser

This was an experiment in Sinatra, a Web Application Framework for Ruby, which I completed as I was finishing Learn Ruby the Hard Way and just before I expressed interest in Founders and Coders. I followed a tutorial from JumpStartLabs.

WhatsApp analyser

"Wow Ivo, this could really save a relationship"

The WhatsApp analyser is the most ambitious project I've worked on, by far. It takes a WhatsApp chat exported to a .txt file (press the three vertically stacked dots, then 'more', then 'export chat'), parses it, analyses it according to 17 metrics, and then presents the data in a GUI containing a table, graphs and a bar-chart. It allows parameterisation, which is where the most interesting programming challenges came from. Specific methods were like Codewars katas of various levels. It's working, but unfinished! Here's a roadmap for where it could go in the future:

  • Running it currently requires both CLI and GUI. I could completely transition to GUI.
  • It doesn't currently use Bundler (a dependency management gem) but does requires the gem Flammarion.
  • There are more graphable metrics to be added.
  • I know of at least one WhatsApp chat file for which the parser throws an error.
  • It's currently very long (500+ lines). I think it's as DRY as I can make it, but I would like to separate out component parts into different files and modules
